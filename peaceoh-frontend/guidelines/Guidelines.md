# PeaceOh — Design Guidelines

## Aesthetic Stance
Soft journaling / mindful-play. Every surface breathes — rounded corners (16–32px), no hard edges, pastel grounds, and gentle motion. Think cozy digital sketchbook, not SaaS dashboard.

## Color Tokens
| Token | Hex | Role |
|---|---|---|
| blush | `#F7C5C5` | CTAs, primary fills |
| lavender | `#D8C9F0` | Section backgrounds, mandala |
| mint | `#C3E8D8` | Sand drawing section |
| buttercup | `#FBF0C2` | Letters section |
| sky | `#C5DFF7` | Pop Bubbles section |
| cream | `#FAF7F2` | Page background |
| text-deep | `#3D3450` | All body text |
| muted | `#8A7FA0` | Captions, placeholders |

## Typography
- **Display** — Playfair Display, italic bold — hero headings only
- **Body/UI** — DM Sans — all interface copy
- **Accent** — Caveat — labels, scores, handwritten moments (use sparingly)

## Layout Principles
- Global border-radius: 16–32px
- Shadows: `0 4px 24px rgba(0,0,0,0.06)` rest, `0 14px 38px rgba(0,0,0,0.13)` hover
- Card hover: `translateY(-6px)` + deeper shadow
- No sharp dividers — use section color changes
- Hide scrollbars by default

## Signature Feature
Bubble cursor trail on home hero only. Pastel circles trail cursor, float up 70px, fade. Brand personality: ephemeral, joyful.
